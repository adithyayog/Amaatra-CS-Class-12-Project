This is the folder for the GUI of the Admin Mode
-->System Requirements:
    >Must have MySQL version 8 and above
    >Python Installation version 3 and above (Recomended)

-->Steps to run the program on an uninitiated computer ( a computer that has never run this program before) :
    >First Download and Run Create Database.exe
    >Second Download and Run Hospital Database Management System.exe
    
-->Steps to run the program on an initiated computer ( a computer that has already run Create Database.exe):
    >First Drop database Project and create new database project
    >Next Download and Run Create Database.exe
    >Last Download and Run Hospital Database Management System.exe
